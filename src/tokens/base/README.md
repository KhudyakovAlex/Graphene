# Base Tokens

Базовые токены - это сырые значения дизайна.

Здесь лежат палитра, размеры, отступы, радиусы, тени, z-index и другие значения без привязки к конкретному месту интерфейса.

`colors.css` содержит базовую цветовую матрицу из Pixso.
Значения разделены по режимам `dark` и `light`.

`radius.css` содержит базовые значения скруглений.
Основной радиус для большинства элементов - `--g-radius-none`.
Остальные радиусы используются точечно, если это задано дизайном или правилом компонента.

`typography.css` содержит базовые значения шрифта, размеров текста, насыщенности и высоты строки.

`spacing.css` содержит базовые расстояния для `padding`, `margin` и `gap`.

`size.css` содержит базовые размеры иконок, контролов и минимальной touch-зоны.

`border.css` содержит базовые толщины границ.

`opacity.css` содержит базовые значения прозрачности.

`z-index.css` содержит базовые значения слоев интерфейса.

Компоненты не должны использовать базовые цвета напрямую, если есть подходящий семантический токен.
