# Icons

В этой папке лежит отобранный набор иконок Graphene.

Правила:

- хранить иконки локально, без загрузки из внешнего интернета;
- добавлять сюда только иконки, утвержденные для дизайн-системы;
- использовать простые имена файлов в kebab-case, например `search.svg`;
- не смешивать основной набор с запасным набором из `others/`.

`others/` хранит большой фиксированный набор иконок на случай редких сценариев.
