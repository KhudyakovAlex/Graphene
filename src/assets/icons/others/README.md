# Other Icons

Здесь хранится большой фиксированный запасной набор иконок.

Иконки из этой папки не считаются основным отобранным набором Graphene.
Если иконка становится регулярной частью дизайн-системы, ее нужно перенести в `src/assets/icons/`.
SVG-иконки должны использовать `currentColor`, чтобы цвет задавался через CSS и semantic tokens.
