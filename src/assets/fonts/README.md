# Fonts

В этой папке должны лежать локальные файлы шрифта IBM Plex Sans.

Graphene не должна загружать шрифты из внешнего интернета.
Использовать можно только файлы, поставляемые вместе с проектом, или утвержденный внутренний источник организации.

Сейчас используются variable TTF файлы:

- `IBMPlexSans-VariableFont_wdth,wght.ttf`
- `IBMPlexSans-Italic-VariableFont_wdth,wght.ttf`

Если позже появятся `woff2`, их можно использовать вместо TTF.

Подключение шрифта описывается в `src/styles/fonts.css`.
