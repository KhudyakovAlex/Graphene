# Graphene Examples

`examples` - витрина Graphene.

Она нужна для демонстрации компонентов и типовых экранов дизайн-системы.

Правила:

- не хранить здесь копии токенов, стилей или компонентов;
- импортировать все элементы дизайн-системы из `src/`;
- использовать `examples/src/views/` для страниц витрины;
- использовать `examples/src/example-data/` только для демонстрационных данных;
- держать витрину работоспособной без внешнего интернета.

`examples/index.html` - точка входа offline-витрины.
