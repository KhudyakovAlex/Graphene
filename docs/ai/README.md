# AI Docs

Инструкции для AI-агентов.

`usage-guide.md`

Читать, когда AI использует Graphene при создании интерфейса приложения.

`development-guide.md`

Читать, когда AI разрабатывает или изменяет саму Graphene.

Если меняются правила Graphene, нужно обновлять оба документа, если изменение влияет и на использование, и на разработку.

Короткая памятка по `Pixso MCP`:

1. Для токенов сначала использовать `get_variables`.
2. Если `get_variables` зависает, можно использовать `get_variable_sets` как fallback.
3. При fallback использовать только локальные и не удаленные переменные.
4. Привязку `semantic token -> base token` определять только по ссылкам на variable id из ответа MCP, а не по визуальной догадке.
