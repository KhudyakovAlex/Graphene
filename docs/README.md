# Graphene Docs

Документация Graphene разделена по аудитории.

`human/`

Документы для людей: объяснение Graphene, структура проекта, правила реализации, шрифт, offline/intranet и витрина.

`ai/`

Инструкции для AI-агентов: как использовать Graphene в приложениях и как разрабатывать саму Graphene.

Если правило важно для единообразия интерфейсов, его нужно фиксировать в соответствующем разделе.
