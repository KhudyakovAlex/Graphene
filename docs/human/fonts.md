# Fonts

В Graphene используется один основной шрифт: IBM Plex Sans.

Файлы шрифта должны лежать в `src/assets/fonts/`.
Подключение через `@font-face` должно находиться в `src/styles/fonts.css`.

Текущий вариант использует variable font файлы:

- обычное начертание: `IBMPlexSans-VariableFont_wdth,wght.ttf`;
- italic: `IBMPlexSans-Italic-VariableFont_wdth,wght.ttf`.

Запрещено:

- подключать шрифты из внешнего интернета;
- использовать CDN;
- добавлять другой основной шрифт без отдельного решения владельца дизайн-системы;
- задавать семейство шрифта напрямую в компонентах, если есть токен или глобальный стиль.

Компоненты должны наследовать шрифт из глобальных стилей.
Если компоненту нужен особый размер, насыщенность или межстрочный интервал, это должно выражаться через типографические токены.
