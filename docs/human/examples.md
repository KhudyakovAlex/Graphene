# Examples

`examples` - витрина Graphene.

Она показывает, как использовать токены, стили и компоненты дизайн-системы.

`examples` не является отдельной дизайн-системой.
Внутри нее нельзя создавать копии токенов, глобальных стилей или компонентов из `src/`.

Правильная зависимость:

`examples` использует `src`.

Неправильная зависимость:

`src` использует `examples`.

В `examples` могут находиться:

- Vite `index.html`;
- Vue-код витрины;
- страницы примеров;
- демонстрационные данные;
- локальные стили самой витрины, если они не подменяют Graphene.

Витрина поставляется вместе с репозиторием как локальный статический сайт.
Основной способ проверки готовой витрины - открыть файл напрямую в браузере:

`examples/index.html`

Для этого не нужен dev-сервер.

Vite dev-сервер используется только во время разработки:

`cd examples && npm run dev`

После изменения Vue-кода витрины нужно обновить локальный bundle для `index.html`:

`cd examples && npm run build`

Страницы примеров должны показывать правильное применение Graphene.
Они не должны становиться бизнес-кодом приложений.
