# Human Docs

Документы для людей, которые впервые знакомятся с Graphene или участвуют в ее развитии.

Основные документы:

- `graphene-for-beginners.md` - короткое объяснение Graphene и структуры проекта.
- `implementation-rules.md` - правила реализации компонентов и токенов.
- `fonts.md` - правила использования IBM Plex Sans.
- `offline-intranet.md` - ограничения для приложений без внешнего интернета.
- `examples.md` - правила устройства витрины Graphene.
