# Offline And Intranet

Приложения на Graphene должны работать без доступа к внешнему интернету.

Разрешено использовать:

- локальные файлы приложения;
- ресурсы внутреннего интранета организации;
- зависимости, поставляемые вместе с приложением или через утвержденный внутренний контур.

Запрещено использовать:

- CDN;
- внешние шрифты;
- внешние иконки;
- внешние изображения;
- публичные API;
- загрузку runtime-зависимостей из интернета.

Если компоненту нужен ассет, он должен быть частью проекта или внутреннего пакета.
